SylLab — System Architecture
Architecture & System Design Document | Version 1.0
1. Tech Stack
The following technology decisions were finalized after Week 1 evaluation of project requirements. Each choice is justified based on the forensic analysis nature of the platform.
1.1 Redis Use Cases
Rate Limiting: Token bucket per instructor for Gemini API (10 req/min). Key pattern: ratelimit:gemini:{instructorId}
Cohort Statistics Cache: Aggregated percentile/stddev per course+week. Key: cohort:stats:{courseId}:{week}. TTL: 3600s
LLM Response Cache: Interview questions per submission. Key: llm:guidance:{submissionId}. TTL: 86400s
Session Store: Active proctored session state. Key: session:proctored:{sessionId}. TTL: session duration + 30min
BullMQ Job Queue (Extra Credit): async AST analysis jobs. Queue name: submission-analysis

3. Environment Strategy
All configuration is validated at startup using Zod. The application refuses to start if any required variable is missing or invalid — preventing 'hello world' placeholder secrets in production.
3.1 Environment Variables (.env structure)
# === Server ===
NODE_ENV=development              # development | production | test
PORT=3000
# === Database ===
DATABASE_URL=postgresql://syllab_user:password@localhost:5432/syllab_db
# === Redis ===
REDIS_URL=redis://localhost:6379
# === JWT ===
JWT_SECRET=<min-32-char-random-string>
JWT_EXPIRES_IN=15m
JWT_REFRESH_SECRET=<different-min-32-char-string>
JWT_REFRESH_EXPIRES_IN=7d
# === Gemini API ===
GEMINI_API_KEY=<from-google-ai-studio>
GEMINI_RATE_LIMIT_PER_MIN=10
# === File Upload ===
MAX_FILE_SIZE_MB=5
# === Cohort Cache ===
COHORT_CACHE_TTL_SECONDS=3600
# === Scoring Thresholds ===
THRESHOLD_MONITOR=1.5
THRESHOLD_FLAG=2.0
THRESHOLD_CRITICAL=2.5
3.2 Zod Validation Schema (src/config/env.ts)
import { z } from 'zod';
const envSchema = z.object({
NODE_ENV: z.enum(['development', 'production', 'test']),
PORT: z.coerce.number().default(3000),
DATABASE_URL: z.string().url(),
REDIS_URL: z.string().url(),
JWT_SECRET: z.string().min(32, 'JWT_SECRET must be at least 32 characters'),
JWT_EXPIRES_IN: z.string().default('15m'),
JWT_REFRESH_SECRET: z.string().min(32),
JWT_REFRESH_EXPIRES_IN: z.string().default('7d'),
GEMINI_API_KEY: z.string().min(10),
GEMINI_RATE_LIMIT_PER_MIN: z.coerce.number().default(10),
MAX_FILE_SIZE_MB: z.coerce.number().default(5),
COHORT_CACHE_TTL_SECONDS: z.coerce.number().default(3600),
THRESHOLD_MONITOR: z.coerce.number().default(1.5),
THRESHOLD_FLAG: z.coerce.number().default(2.0),
THRESHOLD_CRITICAL: z.coerce.number().default(2.5),
});
export const env = envSchema.parse(process.env);
// If any variable is missing/invalid: ZodError thrown at startup — app refuses to start
